# Telegram Checker Bot

A Telegram bot built with Python featuring license key authentication, multi-threaded email validation, and SQLite database management.

## Features
- License key registration system for user authentication
- Multi-threaded mass email checking using ThreadPoolExecutor
- SQLite databases for hits, user queues, and license management
- Admin commands for full bot control

## Tech Stack
- Python
- SQLite
- Telegram Bot API
- python-telegram-bot
